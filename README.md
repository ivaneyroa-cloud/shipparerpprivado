# Shippar Courier - Rastreo Inteligente

Sistema de rastreo de envíos premium desarrollado con **Next.js**, **Tailwind CSS** y **Framer Motion**.

## Características
- 🚀 **Rastreo en Tiempo Real**: Visualización clara del progreso de tus envíos.
- 🎨 **Diseño Premium**: Interfaz moderna con estética de vanguardia (Glassmorphism, Outfit Font).
- 📱 **Responsive**: Optimizado para dispositivos móviles y escritorio.
- ⚡ **Veloz**: Entregas internacionales de 5-8 días.

## Tecnologías
- Next.js 15
- Tailwind CSS
- Framer Motion (Animaciones)
- Lucide React (Iconos)

## Despliegue en Vercel (Paso a Paso)

Sigue estos 3 pasos simples para tener tu app online:

### 1. Sube el código a tu GitHub
Abre tu terminal en la carpeta del proyecto y ejecuta estos comandos:

```bash
# Inicializa el repo
git init

# Agrega los archivos
git add .

# Guarda los cambios
git commit -m "Versión Premium con Avión Realista"

# Conecta con tu repositorio (Asegúrate de que el repo exista en tu GitHub)
git remote add origin https://github.com/ivaneyroa-cloud/shippar-courier-web.git

# Empuja el código
git branch -M main
git push -u origin main
```

### 2. Conecta con Vercel
1. Ve a [Vercel.com](https://vercel.com/new).
2. Selecciona tu repositorio `shippar-courier-web`.
3. Haz clic en **Deploy**.

### 3. ¡Listo!
Vercel te dará un link personalizado (ej: `shippar-courier-web.vercel.app`) que podrás compartir con todos.
